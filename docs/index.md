# Welcome to The Gymnasium

Automated pedagogical system for programming exercises with immediate feedback.

## How it works

1. **Fork** the repository.
2. **Clone** your fork.
3. Solve exercises in the `exercises/` folder.
4. Set `SUBMIT = True` in your `solution.py` when ready.
5. **Push** and open a **Pull Request** to see your score on the leaderboard!

Check the [Exercises](exercises.md) page to see the available challenges.
